# 完整實作步驟

## 準備工作

### 所需工具

1. **Node.js** (18.0+)
   - 下載：https://nodejs.org
   - 選擇 LTS 版本
   - 安裝後驗證：`node -v`

2. **Git**
   - 下載：https://git-scm.com
   - 驗證：`git --version`

3. **GitHub 帳號**
   - 註冊：https://github.com

4. **文字編輯器**
   - 推薦：VS Code (https://code.visualstudio.com)

---

## 階段1：建立 Astro 網站（30分鐘）

### 1.1 建立專案

開啟終端機（Terminal / PowerShell）：

```bash
# 建立新專案
npm create astro@latest my-portfolio

# 互動式選項：
# ✔ Where should we create your new project? › my-portfolio
# ✔ How would you like to start your new project? › Use blog template
# ✔ Install dependencies? › Yes
# ✔ Do you plan to write TypeScript? › No (或 Yes，看你偏好)
# ✔ Initialize a new git repository? › Yes
```

### 1.2 啟動本地伺服器

```bash
cd my-portfolio
npm run dev
```

開啟瀏覽器訪問：http://localhost:4321

**✅ 檢查點：** 你應該看到一個部落格範例網站

---

## 階段2：客製化內容（1-2小時）

### 2.1 修改網站資訊

編輯 `src/consts.ts`：

```typescript
export const SITE_TITLE = 'Pearl 的創作空間';
export const SITE_DESCRIPTION = '故事、遊戲與畫作的世界';
```

編輯 `astro.config.mjs`：

```javascript
import { defineConfig } from 'astro/config';

export default defineConfig({
  site: 'https://你的域名.com',
  // 之後會用到
});
```

### 2.2 建立內容結構

在 `src/content/` 建立分類：

```bash
src/content/
├── blog/          # 現有的部落格（可改名為 stories）
├── games/         # 新增：遊戲介紹
└── art/           # 新增：畫作集
```

### 2.3 撰寫第一篇文章

建立 `src/content/blog/my-first-story.md`：

```markdown
---
title: '我的第一個故事'
description: '這是一個關於冒險的故事'
pubDate: 'Feb 24 2026'
heroImage: '/blog-placeholder.jpg'
tags: ['奇幻', '短篇']
---

## 開始

這是我的第一個故事...

### 支援 Markdown 格式

- **粗體文字**
- *斜體文字*
- [連結](https://example.com)

### 嵌入圖片

![描述文字](/images/my-image.jpg)

### 嵌入 YouTube 影片

<iframe width="560" height="315" 
  src="https://www.youtube.com/embed/影片ID" 
  frameborder="0" allowfullscreen>
</iframe>

### Bilibili 影片（給中國讀者）

<iframe src="//player.bilibili.com/player.html?bvid=BV號" 
  scrolling="no" border="0" frameborder="no" 
  framespacing="0" allowfullscreen="true" 
  width="560" height="315">
</iframe>
```

### 2.4 嵌入遊戲

把 HTML5 遊戲放在 `public/games/` 資料夾：

```
public/
└── games/
    └── my-game/
        ├── index.html
        ├── game.js
        └── assets/
```

在文章中嵌入：

```markdown
<div class="game-container">
  <iframe 
    src="/games/my-game/index.html" 
    width="800" 
    height="600"
    frameborder="0">
  </iframe>
</div>
```

## 階段3：部署到 Cloudflare Pages（20分鐘）

### 3.1 推送到 GitHub

```bash
# 初始化 git（如果還沒有）
git init

# 加入所有檔案
git add .

# 提交
git commit -m "初始網站設定"

# 在 GitHub 建立新 repository
# 名稱：my-portfolio（或任何名稱）
# 然後執行：

git remote add origin https://github.com/你的帳號/my-portfolio.git
git branch -M main
git push -u origin main
```

### 3.2 部署到 Cloudflare Pages

1. **註冊 Cloudflare**
   - 訪問：https://dash.cloudflare.com/sign-up
   - 使用 email 註冊（無需信用卡）

2. **建立 Pages 專案**
   - 左側選單：**Workers & Pages**
   - 點擊 **Create application**
   - 選擇 **Pages** 標籤
   - 點擊 **Connect to Git**

3. **連接 GitHub**
   - 授權 Cloudflare 訪問 GitHub
   - 選擇你的 repository：`my-portfolio`

4. **設定建置**
   ```
   Project name: my-portfolio
   Production branch: main
   Framework preset: Astro
   Build command: npm run build
   Build output directory: dist
   ```

5. **部署**
   - 點擊 **Save and Deploy**
   - 等待 1-2 分鐘
   - 完成後會得到網址：`https://my-portfolio-xxx.pages.dev`

**✅ 檢查點：** 訪問該網址，確認網站上線

### 3.3 綁定自訂域名（可選）

1. 在 Cloudflare Pages 專案頁面
2. **Custom domains** → **Set up a custom domain**
3. 輸入你的域名：`www.你的域名.com`
4. 按照指示完成 DNS 設定
5. 等待 SSL 憑證生效（約 5-10 分鐘）

## 階段4：整合 Waline 留言系統（30分鐘）

> **📘 完整部署指南：** 詳細步驟請參考 [waline-deployment-guide.md](./waline-deployment-guide.md)

### 4.1 部署 Waline 後端到 Vercel

1. **註冊 Vercel**
   - 訪問：https://vercel.com/signup
   - 用 GitHub 帳號登入

2. **一鍵部署 Waline**
   - 訪問：https://vercel.com/new/clone?repository-url=https://github.com/walinejs/waline/tree/main/example
   - 點擊 **Deploy**
   - 專案名稱：`pearl-waline`（或任何名稱）
   - 等待部署完成
   - 記下網址：`https://pearl-waline.vercel.app`

3. **設定資料庫（PostgreSQL + Neon）**
   
   - 在 Vercel 專案中連接 Neon PostgreSQL
   - **重要：** 資料表不會自動建立，需要手動執行腳本
   - 詳細步驟請參考 [waline-deployment-guide.md](./waline-deployment-guide.md)
   - 執行 `scripts/create-waline-tables.js` 建立資料表

4. **設定環境變數**
   
   在 Vercel Environment Variables 中設定：
   ```
   SECURE_DOMAINS=你的網域.pages.dev
   SITE_NAME=Pearl's Blog
   AUTHOR_EMAIL=你的email
   ```

### 4.2 在 Astro 加入留言組件

建立 `src/components/Comments.astro`：

```astro
---
const { path } = Astro.props;
const walineServer = 'https://pearl-waline.vercel.app'; // 改成你的
---

<div id="waline-container"></div>

<script define:vars={{ path, walineServer }}>
  // 動態載入 Waline
  import('https://unpkg.com/@waline/client@v3/dist/waline.js').then((module) => {
    const Waline = module.default || module;
    Waline.init({
      el: '#waline-container',
      serverURL: walineServer,
      path: path || window.location.pathname,
      lang: 'zh-TW',
      locale: {
        placeholder: '留下你的想法...',
        sofa: '快來搶沙發~',
        submit: '發送',
        nick: '暱稱',
        mail: 'Email',
        link: '網址',
      },
      emoji: [
        '//unpkg.com/@waline/emojis@1.2.0/weibo',
        '//unpkg.com/@waline/emojis@1.2.0/bilibili',
      ],
      dark: 'auto',
      meta: ['nick', 'mail', 'link'],
      requiredMeta: ['nick'],
      pageview: true,
      comment: true,
    });
  });
</script>

<style is:global>
  @import 'https://unpkg.com/@waline/client@v3/dist/waline.css';
  
  #waline-container {
    margin-top: 3rem;
    padding-top: 2rem;
    border-top: 1px solid #e5e7eb;
  }
  
  .wl-panel {
    background: #f9fafb;
    border-radius: 8px;
    padding: 1rem;
  }
</style>
```

### 4.3 整合到文章頁面

找到 `src/layouts/BlogPost.astro`（或類似檔案），加入：

```astro
---
import Comments from '../components/Comments.astro';
// ...其他 imports
---

<html>
  <body>
    <article>
      <!-- 文章內容 -->
      <slot />
    </article>
    
    <!-- 留言區 -->
    <Comments path={Astro.url.pathname} />
  </body>
</html>
```

### 4.4 推送更新

```bash
git add .
git commit -m "整合 Waline 留言系統"
git push
```

Cloudflare Pages 會自動重新部署（1-2 分鐘）

**✅ 檢查點：** 訪問任一文章頁面，應該看到留言區

## 階段5：設定留言管理（10分鐘）

### 5.1 註冊管理員帳號

1. 訪問：`https://pearl-waline.vercel.app/ui/register`
2. 填寫資訊（第一個註冊的自動成為管理員）：
   - Email
   - 密碼
   - 暱稱
   - 網址（可選）

### 5.2 管理留言

之後從 `https://pearl-waline.vercel.app/ui/login` 登入

功能：
- 審核留言
- 刪除垃圾留言
- 查看統計數據
- 匯出留言

### 5.3 郵件通知設定（可選）

在 Vercel 專案環境變數加入：

```
SMTP_SERVICE=Gmail
SMTP_USER=你的email@gmail.com
SMTP_PASS=你的Gmail應用程式密碼
SITE_NAME=Pearl 的創作空間
SITE_URL=https://你的域名.com
AUTHOR_EMAIL=你的email@gmail.com
```

**如何取得 Gmail 應用程式密碼：**
1. Google 帳戶安全性設定
2. 開啟「兩步驟驗證」
3. 「應用程式密碼」→ 產生新密碼
4. 複製 16 位密碼（不含空格）

重新部署 Vercel 專案後，有人留言或回覆時會收到通知信

## 日常使用流程

### 發佈新文章

```bash
# 1. 建立新檔案
touch src/content/blog/new-story.md

# 2. 編輯內容（用你喜歡的編輯器）

# 3. 本地預覽
npm run dev

# 4. 推送上線
git add .
git commit -m "新文章：標題"
git push

# 5. 等待 1-2 分鐘自動部署
```

### 更新網站設計

```bash
# 1. 修改樣式或版型
# 編輯 src/styles/ 或 src/layouts/

# 2. 本地測試
npm run dev

# 3. 滿意後推送
git add .
git commit -m "更新網站設計"
git push
```

### 管理留言

- 訪問：`https://pearl-waline.vercel.app/ui`
- 定期檢查並審核留言
