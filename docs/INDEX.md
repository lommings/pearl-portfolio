## 📁 專案結構

```
pearl-portfolio/
├── docs/              ← 你現在在這裡（文檔）
│   ├── INDEX.md      ← 本文件（導航）
│   ├── README.md     ← 專案總覽
│   ├── 實作步驟.md
│   ├── 程式碼範例.md
│   ├── 疑難排解.md
│   ├── 快速參考.md
│   └── 建站檢查清單.md
├── src/              ← 網站原始碼
│   ├── components/   ← 組件
│   ├── content/blog/ ← 文章（在這裡寫！）
│   ├── layouts/      ← 版型
│   └── pages/        ← 頁面
├── public/           ← 靜態資源（圖片、遊戲）
├── 快速啟動.md       ← 新手必看
├── 使用說明.md       ← 日常使用
└── package.json      ← 專案設定
```

## 🚀 常用指令

```bash
# 啟動開發伺服器
npm run dev

# 建置網站
npm run build

# 預覽建置結果
npm run preview
```
