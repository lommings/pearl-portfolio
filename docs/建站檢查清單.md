# 建站檢查清單

用這份清單追蹤你的進度，完成後打勾 ✅

---

## 📦 準備階段

### 安裝工具

- [ ] 安裝 Node.js (18.0+)
  - 驗證：`node -v`
- [ ] 安裝 Git
  - 驗證：`git --version`
- [ ] 註冊 GitHub 帳號
- [ ] 安裝文字編輯器（VS Code 推薦）
- [ ] 安裝 VS Code 的 Astro 外掛

### 準備帳號

- [ ] GitHub 帳號已建立
- [ ] Cloudflare 帳號已註冊（免費）
- [ ] Vercel 帳號已註冊（用 GitHub 登入）
- [ ] （可選）域名已購買

---

## 🚀 階段1：建立網站

### 建立 Astro 專案

- [ ] 執行 `npm create astro@latest my-portfolio`
- [ ] 選擇 Blog 模板
- [ ] 成功安裝依賴套件
- [ ] 執行 `npm run dev`
- [ ] 能訪問 http://localhost:4321
- [ ] 看到範例部落格網站

### 測試基本功能

- [ ] 能看到範例文章
- [ ] 修改檔案後自動重新載入
- [ ] 圖片能正常顯示

---

## ✏️ 階段2：客製化內容

### 基本設定

- [ ] 修改 `src/consts.ts` 的網站名稱和描述
- [ ] 更新 `astro.config.mjs` 的 site 設定
- [ ] 替換 favicon（`public/favicon.ico`）

### 建立內容

- [ ] 撰寫第一篇測試文章
- [ ] 上傳至少一張圖片到 `public/images/`
- [ ] 測試圖片在文章中顯示
- [ ] （可選）建立 About 頁面

### 組件整合

- [ ] 複製 `Comments.astro` 組件（留言系統）
- [ ] 複製 `VideoPlayer.astro` 組件（影片播放）
- [ ] （可選）複製 `GameEmbed.astro` 組件
- [ ] 測試組件是否正常運作

---

## 🌐 階段3：部署網站

### Git 與 GitHub

- [ ] 執行 `git init`（如果還沒有）
- [ ] 在 GitHub 建立新 repository
- [ ] 執行 `git add .`
- [ ] 執行 `git commit -m "初始提交"`
- [ ] 連接遠端 repo：`git remote add origin ...`
- [ ] 推送：`git push -u origin main`
- [ ] 確認 GitHub 上能看到程式碼

### Cloudflare Pages 部署

- [ ] 登入 Cloudflare Dashboard
- [ ] Workers & Pages → Create application → Pages
- [ ] Connect to Git → 授權 GitHub
- [ ] 選擇你的 repository
- [ ] 設定建置：
  - Framework preset: Astro
  - Build command: `npm run build`
  - Build output: `dist`
- [ ] Save and Deploy
- [ ] 等待建置完成（1-2 分鐘）
- [ ] 訪問 `https://xxx.pages.dev` 確認網站上線
- [ ] 測試所有頁面和功能

### （可選）自訂域名

- [ ] Cloudflare Pages → Custom domains
- [ ] 新增你的域名
- [ ] 設定 DNS 記錄（Cloudflare 通常自動完成）
- [ ] 等待 SSL 憑證生效（5-15 分鐘）
- [ ] 訪問自訂域名確認正常

---

## 💬 階段4：整合留言系統

### 部署 Waline 後端

- [ ] 登入 Vercel（用 GitHub 帳號）
- [ ] 一鍵部署 Waline：  
      https://vercel.com/new/clone?repository-url=https://github.com/walinejs/waline/tree/main/example
- [ ] 專案名稱：`pearl-waline`（或自訂）
- [ ] 等待部署完成
- [ ] 記下 Waline 網址：`https://pearl-waline.vercel.app`
- [ ] 訪問該網址確認 Waline 運作中

### （可選）設定 LeanCloud 儲存

- [ ] 註冊 LeanCloud：https://console.leancloud.cn
- [ ] 建立應用
- [ ] 取得 App ID, App Key, Server URL
- [ ] 在 Vercel 設定環境變數：
  - `LEAN_ID`
  - `LEAN_KEY`
  - `LEAN_SERVER`
- [ ] 重新部署 Vercel 專案

### 整合到 Astro

- [ ] 建立 `src/components/Comments.astro`
- [ ] 更新 `serverURL` 為你的 Waline 網址
- [ ] 在 `BlogPost.astro` 加入 `<Comments />` 組件
- [ ] 本地測試：`npm run dev`
- [ ] 確認留言區顯示
- [ ] 測試發送留言功能
- [ ] Git 提交並推送
- [ ] 等待 Cloudflare 自動部署
- [ ] 在正式網站上測試留言功能

---

## ⚙️ 階段5：設定管理

### Waline 管理後台

- [ ] 訪問：`https://你的waline.vercel.app/ui/register`
- [ ] 註冊管理員帳號
- [ ] 登入管理後台
- [ ] 測試審核留言功能
- [ ] 設定網站名稱和資訊

### 郵件通知（可選）

- [ ] 取得 Gmail 應用程式密碼：
  - Google 帳戶 → 安全性
  - 開啟兩步驟驗證
  - 生成應用程式密碼
- [ ] 在 Vercel 設定環境變數：
  - `SMTP_SERVICE=Gmail`
  - `SMTP_USER=你的email`
  - `SMTP_PASS=應用程式密碼`
  - `AUTHOR_EMAIL=你的email`
  - `SITE_NAME=網站名稱`
  - `SITE_URL=網站網址`
- [ ] 重新部署 Vercel
- [ ] 測試：發送留言後檢查是否收到信
