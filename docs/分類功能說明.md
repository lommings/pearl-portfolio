# 📂 分類與標籤功能使用說明

## ✨ 新功能

你的網站現在支援：
- ✅ **左側選單** - 自動按分類顯示所有文章
- ✅ **分類管理** - 每篇文章可以指定一個分類
- ✅ **標籤系統** - 每篇文章可以有多個標籤
- ✅ **熱門標籤** - 側邊欄顯示最常用的標籤
- ✅ **最新文章** - 快速查看最近發表的內容
- ✅ **標籤頁面** - 點擊標籤查看相關文章

---

## 📝 如何使用

### 1. 撰寫文章時加入分類和標籤

在文章的 frontmatter 中加入：

```markdown
---
title: '我的故事'
description: '這是一個冒險故事'
pubDate: 'Feb 24 2026'
heroImage: '../../assets/blog-placeholder-1.jpg'
category: '奇幻小說'         # 分類（只能一個）
tags: ['冒險', '魔法', '友情']  # 標籤（可以多個）
---

故事內容...
```

### 2. 分類規則

**分類（category）：**
- 每篇文章只能有**一個**分類
- 用來組織文章的主要類型
- 會顯示在左側選單中
- 建議分類：
  - 故事類：`奇幻小說`、`科幻小說`、`短篇故事`、`連載小說`
  - 遊戲類：`RPG遊戲`、`解謎遊戲`、`視覺小說`
  - 畫作類：`插畫`、`漫畫`、`設定圖`
  - 其他：`教學`、`日記`、`公告`

**標籤（tags）：**
- 每篇文章可以有**多個**標籤
- 用來標記文章的關鍵詞
- 可以點擊查看相同標籤的文章
- 建議標籤：
  - 主題：`冒險`、`愛情`、`懸疑`、`搞笑`
  - 元素：`魔法`、`機器人`、`時間旅行`
  - 情感：`溫馨`、`熱血`、`感動`

## 🖼️ 在文章中插入圖片

### 方法1：使用 public 資料夾（簡單）

1. 把圖片放到 `public/images/` 資料夾
2. 在 Markdown 中使用：

```markdown
![圖片說明](/images/my-photo.jpg)
```

### 方法2：使用 assets 資料夾（推薦，自動優化）

1. 把圖片放到 `src/assets/` 資料夾
2. 作為 heroImage 使用：

```markdown
---
heroImage: '../../assets/my-cover.jpg'
---
```

3. 在文章中嵌入：

```markdown
import { Image } from 'astro:assets';
import myImage from '../../assets/my-image.jpg';

<Image src={myImage} alt="圖片說明" width={800} />
```

### 方法3：圖片畫廊（多張圖）

```markdown
## 作品展示

![作品1](/images/art-1.jpg)
![作品2](/images/art-2.jpg)
![作品3](/images/art-3.jpg)
```

**圖片最佳實踐：**
- 使用 WebP 或 JPG 格式
- 建議寬度：800-1200px
- 壓縮圖片（使用 [TinyPNG](https://tinypng.com)）
- 加上有意義的 alt 文字

## 🎨 側邊欄功能

### 自動顯示內容：

1. **📂 分類** - 按分類分組顯示所有文章
   - 可以展開/收合
   - 顯示每個分類的文章數量
   - 當前文章會高亮顯示

2. **🏷️ 熱門標籤** - 顯示最常用的前 10 個標籤
   - 點擊可查看該標籤的所有文章
   - 顯示每個標籤使用次數

3. **📝 最新文章** - 最近發表的 5 篇文章
   - 顯示標題和日期
   - 當前文章會高亮顯示

## 💡 使用建議

### 如何規劃分類：

1. **不要建立太多分類**
   - 建議 5-10 個分類
   - 太多會讓側邊欄太長

2. **分類要互斥**
   - 一篇文章應該明確屬於一個分類
   - 例如：`小說` vs `遊戲` vs `畫作`

3. **使用有意義的名稱**
   - ✅ `奇幻小說`、`RPG遊戲`
   - ❌ `類型1`、`其他`

### 如何使用標籤：

1. **標籤要具體**
   - ✅ `時間旅行`、`機器人`、`推理`
   - ❌ `好看`、`很棒`

2. **每篇文章 3-5 個標籤最佳**
   - 太少：不容易被找到
   - 太多：失去重點

3. **建立標籤規範**
   - 統一用詞（`科幻` vs `SF`）
   - 避免重複（`魔法` vs `魔術`）

## 🛠️ 常見問題

### Q: 不想用分類，可以嗎？

A: 可以！不寫 `category` 欄位，文章會自動歸類到「未分類」。

### Q: 標籤可以有空格嗎？

A: 可以，但建議用短詞：
```markdown
tags: ['時間旅行', '平行宇宙']  # ✅ 可以
tags: ['這是一個很長的標籤名稱']  # ❌ 不建議
```

### Q: 可以修改已發表文章的分類嗎？

A: 可以！直接編輯 `.md` 檔案的 frontmatter，儲存後側邊欄會自動更新。

### Q: 側邊欄太長怎麼辦？

A: 側邊欄會自動出現滾動條，而且是固定位置（sticky），滾動時不會消失。
