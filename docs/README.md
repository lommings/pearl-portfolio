## 💡 文檔使用技巧

1. **先看目錄** - INDEX.md 有完整導航
2. **善用搜尋** - 用 Ctrl+F 搜尋關鍵字
3. **逐步跟著做** - 不要跳步驟
4. **遇到問題先查疑難排解** - 大部分問題都有答案
5. **保持文檔更新** - 有新經驗就加進來
