# Pearl 個人網站建置計畫

## 📋 專案概述

建立一個個人創作網站，用於發表：
- 📝 故事作品
- 🎮 小遊戲
- 🎨 畫作
- 🎬 YouTube/Bilibili 影片

## 🎯 核心需求

- ✅ 中國用戶可訪問
- ✅ 支援會員系統（未來功能）
- ✅ 讀者留言互動
- ✅ 靈活新增頁面
- ✅ 嵌入 YouTube/Bilibili 影片
- ✅ 免費或低成本

## 🛠️ 技術架構

### 前端網站
- **框架：** Astro (靜態網站生成器)
- **部署：** Cloudflare Pages
- **域名：** 你現有的域名
- **成本：** 免費

### 留言系統
- **服務：** Waline
- **部署：** Vercel
- **成本：** 免費

### 會員系統（第二階段）
- **服務：** Supabase
- **部署：** Supabase 雲端
- **成本：** 免費額度內

## ⏱️ 預估時間

- **階段1：建立網站** - 30 分鐘
- **階段2：客製化內容** - 1-2 小時
- **階段3：部署上線** - 20 分鐘
- **階段4：整合留言** - 30 分鐘
- **階段5：設定管理** - 10 分鐘

**總計：約 3-4 小時**（首次設定）

## 💰 成本預估

- Astro 網站：**免費**（Cloudflare Pages）
- Waline 留言：**免費**（Vercel）
- 域名：**約 $10-15/年**（已有則跳過）
- **總計：可完全免費運行！**

## 🚀 快速開始

```bash
# 1. 建立專案
npm create astro@latest my-portfolio

# 2. 進入目錄
cd my-portfolio

# 3. 啟動開發伺服器
npm run dev

# 4. 開啟瀏覽器
# http://localhost:4321
```
