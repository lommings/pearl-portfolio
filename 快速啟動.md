## 🎯 下一步

### 1. 本地預覽網站（馬上試試看！）

```bash
cd C:\Users\lommi\Projects\pearl-portfolio
npm run dev
```

然後開啟瀏覽器訪問：**http://localhost:4321**

### 2. 開始撰寫你的第一篇文章

1. 在 `src/content/blog/` 建立新檔案，例如：`my-first-story.md`

2. 複製這個範本：

```markdown
---
title: '我的第一個故事'
description: '這是一個關於...的故事'
pubDate: 'Feb 24 2026'
heroImage: '/blog-placeholder-1.jpg'
---

## 故事開始

從前從前...
```

3. 儲存後，網站會自動更新！

### 3. 部署 Waline 留言系統

**步驟：**

1. 訪問：https://vercel.com/new/clone?repository-url=https://github.com/walinejs/waline/tree/main/example

2. 用 GitHub 帳號登入

3. 點擊 **Deploy**

4. 記下你的 Waline 網址：`https://你的專案.vercel.app`

5. 編輯 `src/components/Comments.astro`，把這一行：
   ```javascript
   const walineServer = 'https://your-waline.vercel.app';
   ```
   改成你的 Waline 網址

### 4. 推送到 GitHub

**如果你還沒有 GitHub repo：**

1. 去 https://github.com/new 建立新 repository
2. 名稱：`pearl-portfolio`（或任何你喜歡的）
3. 不要勾選任何選項（README, .gitignore 等）
4. Create repository

**然後執行：**

```bash
cd C:\Users\lommi\Projects\pearl-portfolio
git remote add origin https://github.com/你的帳號/pearl-portfolio.git
git push -u origin master
```

### 5. 部署到 Cloudflare Pages

1. 登入：https://dash.cloudflare.com

2. **Workers & Pages** → **Create application** → **Pages**

3. **Connect to Git** → 選擇你的 `pearl-portfolio` repo

4. 建置設定：
   - Framework preset: **Astro**
   - Build command: `npm run build`
   - Build output directory: `dist`

5. **Save and Deploy**

6. 等待 1-2 分鐘，你的網站就上線了！🎉

## 📝 日常使用

### 新增文章

```bash
# 1. 建立新檔案
src/content/blog/my-new-story.md

# 2. 撰寫內容（用任何文字編輯器）

# 3. 本地預覽
npm run dev

# 4. 滿意後推送
git add .
git commit -m "新增文章：標題"
git push

# 5. Cloudflare 會自動部署（1-2 分鐘）
```

### 嵌入影片

在文章中加入：

```markdown
import VideoPlayer from '../../components/VideoPlayer.astro';

<VideoPlayer 
  youtube="dQw4w9WgXcQ"
  bilibili="BV1xx411c7XZ"
  title="影片標題"
/>
```

### 嵌入遊戲

1. 把遊戲檔案放到 `public/games/my-game/`

2. 在文章中：

```markdown
import GameEmbed from '../../components/GameEmbed.astro';

<GameEmbed 
  src="/games/my-game/index.html"
  title="我的遊戲"
/>
```
