# Pearl 個人網站使用說明

## 🚀 快速開始

### 本地開發

```bash
# 進入專案目錄
cd pearl-portfolio

# 啟動開發伺服器
npm run dev

# 開啟瀏覽器訪問 http://localhost:4321
```

### 建置網站

```bash
# 建置生產版本
npm run build

# 預覽建置結果
npm run preview
```

---

## 📝 撰寫文章

### 1. 建立新文章

在 `src/content/blog/` 資料夾建立新的 `.md` 檔案，例如：`my-story.md`

### 2. 文章格式

```markdown
---
title: '文章標題'
description: '文章摘要（會顯示在列表）'
pubDate: 'Feb 24 2026'
heroImage: '/blog-placeholder-1.jpg'
---

## 這是文章內容

你的故事從這裡開始...
```

### 3. 使用圖片

**方法1：放在 public 資料夾**
1. 把圖片放到 `public/images/`
2. 在文章中使用：`![說明](/images/my-image.jpg)`

**方法2：使用 Astro Image 組件**（推薦，自動優化）
1. 把圖片放到 `src/assets/`
2. 在文章中使用 Image 組件

## 🎬 嵌入影片

### YouTube 和 Bilibili

在文章中使用 VideoPlayer 組件：

```markdown
import VideoPlayer from '../../components/VideoPlayer.astro';

<VideoPlayer 
  youtube="影片ID"
  bilibili="BV號"
  title="影片標題"
/>
```

**如何取得影片 ID：**
- YouTube: `https://www.youtube.com/watch?v=dQw4w9WgXcQ` → ID 是 `dQw4w9WgXcQ`
- Bilibili: `https://www.bilibili.com/video/BV1xx411c7XZ` → BV 號是 `BV1xx411c7XZ`

---

## 🎮 嵌入遊戲

### HTML5 遊戲

1. 把遊戲檔案放到 `public/games/` 資料夾
2. 在文章中使用：

```markdown
import GameEmbed from '../../components/GameEmbed.astro';

<GameEmbed 
  src="/games/my-game/index.html"
  title="我的遊戲"
/>
```

## 💬 留言系統設定

### 部署 Waline 後端

1. 訪問：https://vercel.com/new/clone?repository-url=https://github.com/walinejs/waline/tree/main/example
2. 用 GitHub 登入 Vercel
3. 點擊 Deploy
4. 記下 Waline 網址：`https://你的專案.vercel.app`

### 更新留言組件

編輯 `src/components/Comments.astro`：

```javascript
const walineServer = 'https://你的專案.vercel.app'; // 改成你的 Waline 網址
```

## 🌐 部署到 Cloudflare Pages

### 1. 推送到 GitHub

```bash
git add .
git commit -m "初始提交"
git push origin main
```

### 2. 連接 Cloudflare Pages

1. 登入 https://dash.cloudflare.com
2. Workers & Pages → Create application → Pages
3. Connect to Git → 選擇你的 repository
4. 建置設定：
   - Framework preset: **Astro**
   - Build command: `npm run build`
   - Build output directory: `dist`
5. Save and Deploy

### 3. 綁定自訂域名（可選）

1. Cloudflare Pages → Custom domains
2. 新增你的域名
3. 設定 DNS（Cloudflare 通常自動完成）

## 🛠️ 常見問題

### 本地開發伺服器無法啟動

```bash
# 刪除 node_modules 重新安裝
rm -rf node_modules package-lock.json
npm install
```

### 圖片無法顯示

- 確認圖片在 `public/` 資料夾
- 路徑要以 `/` 開頭，例如：`/images/photo.jpg`

### Markdown 格式錯誤

- frontmatter（`---` 之間的部分）必須使用正確的 YAML 格式
- 標題、描述要用引號包起來
- 日期格式：`'Feb 24 2026'` 或 `'2026-02-24'`
